```
装甲架  
工坊名：Armor Racks  
分类：功能机制  
汉化：有  
正义度：正义  
内容：这个模组新增加了几个装甲架，小人可以把衣服放在上面，需要的时候一键换装。据说配合整装待命很好用，但楼主不会用  
正义的地方在于有个电力修复盔甲架，可以修复放在上面的装备，再也不用担心装备报废了  
要注意的是这个模组好像和pick up and haul有一点小冲突，小人可能会在盔甲架前面卡住，解决办法是在非战斗时手动把盔甲架禁止（F）掉，开战的时候F+强制命令换装即可  
下面再推荐几个同类型的
```

```
修理与回收  
工坊名：MendAndRecycle  
分类：功能机制  
汉化：有  
正义度：有一点正义  
内容：这个mod新增了一个修补工作台，需要科技研发解锁。可以在修补工作台上修补你装备的耐久，还可以回收物品（有点像电熔炼器，但是可以回收布料）。通常来说修东西需要“修理工具包”，得在加工台造，修补台还耗电。  
这个mod的修理/回收是在工作台上操作的，另外它明确的和边缘工厂那个自动工作站不兼容（似乎和配方库也不兼容，但楼主没一起用过）；这个mod的好处在于机械体也能回收/修理物品了
```


```
修理物品  
工坊名：repair items  
分类：功能机制  
汉化：有（1.3，但是可以用）  
正义度：有一点正义  
内容：这个模组允许小人维修在储存区里摆着的装备（mod配置里好像还可以改维修仿生体、材料啥的，不过楼主只用它维修装备）；你的小人会操着电焊枪上去维修。  
这个mod的好处在于，你可以通过设置服装方案来达成服装自动维修（在方案里设定小人只穿51-100%的服装，这样小人服装耐久低了以后会自己脱到储存区，然后你的制作小人会在有空的时候跑去修。因为rimworld同一个物品在同一时间只会被一个人占用，小人会一直把这衣服修好，然后别人会重新去穿）
```
