```
频带节点强化  
工坊名：Advanced Band Nodes  
分类：机械强化  
汉化：有  
正义度：有一点正义  
内容：你还在犹豫机械师带宽不够用，却又不想打超正义mod吗，那这个mod还比较适合你。这个mod添加了一个需要高科才能解锁的高级频带节点，耗电翻倍，给4个带宽，需要高级零部件。还算比较符合原版的味道
```

```
允许掘进机钻探  
工坊名：Allow tunnelers to drill  
分类：功能机制  
汉化：无需  
正义度：有一点正义  
内容：掘进机大哥很厉害，肉的一比的同时还能挖矿，但是每次把地表矿挖完以后，它就没有矿挖了。这个mod允许掘进机操作深钻井（一个掘进机可以单挑三四个虫子）  
（PS：需要注意的是，这个mod可能和某些基于深钻井机制产生的mod之间产生交互上的bug，比如deep rim：掘进机会卡在建筑前面。不过解决方法也很简单，强制征召掘进机让它待机/休眠，手动命令你的小人完成任务即可）
```
