Misc训练

分类：功能机制  
汉化：合集（就几个建筑，可以不打汉化，因为misc其他的东西现在多半不用了）

```
resource dictionary
resource dictionary就是给每一种东西一个分类，吧两样物品放进同一个类就可以算作同一个东西。例如因为mod你有两种铁矿A和B，A可以直接用于建造，B需要冶炼后才能用，而你不想冶炼，那么只要把B添加到A的分类中，B就可以被视作A，从而可以直接用B建造。如果你玩过MC，可能会知道矿物词典
区别在于resource dictionary对各种物品都有效，并且可以把同类物品全部替换为类型中的主体物品；一般来说resource dictionary默认归类好了大部分物品，少数冷门mod或者并非常识中为一类的物品可以自己手动调整。
比如你可以把边缘工厂的量子核心添加到生物科技DLC的纳米芯片分类中，然后用量子核心来制作垃圾分解器。
但resource dictionary并不能影响物品的特殊功能，例如添加到纳米芯片那类的物品并不能代替纳米芯片解锁极致机械科技，添加到火盆分类的灯也不能代替火盆满足谒见厅的要求。

```
