```
Filter Manager
筛选器管理器
在mod物品比较多的情况下真的很难记清楚哪些东西会腐烂。  
打了这个mod之后，冷库/冰箱我就直接选【预置-会腐烂的】，然后再把人类尸体叉掉，食物全选了就好。  
  
它还有一个反选的功能，前期基础的低优先级存储区就可以设置成【预置-会腐烂的-反选】，再把食物叉掉就行。  
也能保存自定义的设置，比如上面提到的这两种，也可以保存一个【冷库】预置然后反选就行。（还是要叉掉人类尸体）  
具体用法可以自己摸索。
```
